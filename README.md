# CryptoSpin – Online Casino

A modern online casino platform with cryptocurrency integration, built with TypeScript and Next.js.

## Technologies
- TypeScript
- Next.js
- React
- Cryptocurrency APIs

## Features
- User registration and authentication
- Casino games (slots, roulette, etc.)
- Crypto wallet integration
- Real-time balance updates
- Responsive UI

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

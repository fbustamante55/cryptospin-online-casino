Queremos expandir y actualizar nuestro casino físico con un nuevo proyecto online llamado NOVA FORTUNE: un nombre corto, impactante y fácil de recordar, relacionado con "nuevas oportunidades" (Nova = nueva estrella) y "fortuna" (Fortune = suerte, riqueza). Este branding reforzará nuestra identidad premium y moderna, con colores dominantes azul profundo, dorado suave y acentos en blanco, siguiendo un estilo elegante, sofisticado y adaptado a un público internacional.

La plataforma online debe desarrollarse utilizando tecnologías modernas:

Frontend: Next.js

Backend: Node.js con Express.js

Base de datos: PostgreSQL utilizando Prisma ORM para gestión de datos

Web3.js o Ethers.js: para depósitos y retiros en criptomonedas

El sistema debe ser completamente responsive, pensado tanto para desktop como para mobile.

Estructura del proyecto:
/frontend: Sitio web público y de usuarios con páginas como Home, Login, Registro, Dashboard, Depósitos y Retiros.

/backend: APIs REST para gestión de usuarios, billeteras, transacciones, validaciones KYC, integración de juegos y apuestas deportivas.

/database/prisma: Definición de schema.prisma y migraciones de base de datos.

Funcionalidades esenciales:
Sistema de registro/login con KYC obligatorio (captura de nombre, apellidos, fecha de nacimiento, nacionalidad, dirección, teléfono, correo electrónico, documentos de identidad/selfie).

Login seguro mediante tokens JWT.

Usuarios con KYC aprobado podrán depositar, jugar y retirar fondos.

Billetera multicripto por usuario: soporte para balances en BTC, ETH, USDT (ERC-20/TRC-20) y LTC.

Generación de direcciones únicas de depósito por cripto.

Detección automática de depósitos vía blockchain.

Procesamiento manual o automático de retiros, aplicando comisiones configurables.

Integraciones requeridas:
APIs de FrontOffice CMS (registro, login, KYC) y iGaming Platform API (juegos de casino y apuestas deportivas).

Preparado para integrar procesadores de pagos cripto como NOWPayments o Binance Pay.
